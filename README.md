# Task Management System (C++)

A simple console application to manage tasks, projects, and users using OOP principles.

## Features
- Create Users and Projects.
- Add Tasks with Priority (High, Medium, Low).
- Track Task Status (To Do, In Progress, Done).
- Built using C++ Classes and STL (Vectors & Maps).

## How to use
- Compile all `.cpp` files.
- Run the program and follow the console instructions.
